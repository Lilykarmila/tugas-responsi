Pengertian Web Hosting

Hosting adalah layanan berbasis internet yang digunakan sebagai tempat penyimpanan data atau tempat untuk menajalankan aplikasi ditempat terpusat yang disebut sebagai server dan dapat diakses melalui jaringan internet. Jenis penggunaannya bermacam-macam sehingga dapat disesuaikan dengan kebutuhan masing-masing, seperti web untuk penyimpanan dan aplikasi website melalui port http, Email untuk pusat data email, FTP utnuk transfer bahkan sampai VPN.Dengan adanya hosting ini, halaman website yang anda buat dapat diakses dengan oranag lain
